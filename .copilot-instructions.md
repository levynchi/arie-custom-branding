## 🚨 דפוסים אסורים לחלוטין
- `element.style.anything = value`
- `style="background: red"`  
- `<div style="width: 100px">`
- קבועים קשיחים בקוד (30, 100px, וכד')

## ✅ הדרך הנכונה
במקום:
```javascript
element.style.background = 'red';
```

עשה:
```javascript
element.classList.add('error-background');
```

ואז ב-CSS:
```css
.error-background {
    background: red;
}
```
